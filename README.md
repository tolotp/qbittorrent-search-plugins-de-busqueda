IMPORTANTE LEER
===
Para usar estos plugins tienes que tener instalado Python en tu pc, haz click [aquí](https://www.python.org/downloads/) para instalarlo, te recomiendo la version 3.13.x puede ser cualquiera, lo descargas e instalas, en la pestaña de instalación te vendra en la parte de abajo 2 cuadrados para que hagas un "check" ponle un visto a los 2 y le das a instalar.

Para una guia mas completa de como instalar python haz click aqui

**Estos plugins fueron probados en la versión 5.1.4 de qBittorrent, deberían funcionar desde las versiones 4.5.x, 4.6.x en adelante**

Adicionalmente tambien necesitar tener activado el "Motor De Búsqueda" en qbittorrent, este se encuentra en la opción "Ver" y ahí aparecera desactivado, tienes que activarlo haciendole click.

Para instalar estos plugins tienes 2 opciones, puedes hacerlo copiando la url del plugin (es mas recomendable hacerlo de esta manera para recibir actualizaciones en caso de que se necesiten) o descargando el archivo .py
