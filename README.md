IMPORTANTE LEER
===
Para usar estos plugins tienes que tener instalado Python en tu PC, te recomiendo la version 3.13.x puede ser cualquiera de esa versión, ejemplo 3.13.1 / 3.13.2, etc, lo descargas e instalas. Si ya tienes instalado Python pero en otra versión asegurate de que sea 3.x.x para arriba aunque te recomiendo las versiones mas recientes igualmente.

Para una guia mas completa de como instalar Python haz click [AQUÍ](https://github.com/tolotp/qbittorrent-search-plugins-de-busqueda/wiki/Gu%C3%ADa-para-instalar-Python)

**Estos plugins fueron probados en la versión 5.1.4 de qBittorrent, deberían funcionar desde las versiones 4.5.x, 4.6.x en adelante.**

Necesitas tener activado el `Motor De Búsqueda` en qbittorrent, si no lo tienes haz click [AQUÍ](https://github.com/tolotp/qbittorrent-search-plugins-de-busqueda/wiki/Gu%C3%ADa-para-habilitar-el-la-b%C3%BAsqueda-en-qbit) para ver una guía.

Para instalar un plugin correctamente mira esta guia [AQUÍ](https://github.com/tolotp/qbittorrent-search-plugins-de-busqueda/wiki/Gu%C3%ADa-para-Instalar-un-plugin-en-qbit).

Lee con atención la tabla de plugins en la parte de **"Comentarios"** la `Nota:` del plugin para ver si necesitas una guia para instalarlo.
